# Calcul-de-l-IBH
IBH renvoie à Indice de bien être histologique. Il s'agit d'un projet proposé en olympiades de mathématiques avec une adaptation en code. Le projet consiste à calculer le niveau de vie d'un foyer avec 15 variables précises. Le but est d'automatiser la tache avec l'ajout d'un fichier .csv ou excel
